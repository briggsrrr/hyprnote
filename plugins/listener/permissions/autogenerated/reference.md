## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-list-microphone-devices`
- `allow-get-current-microphone-device`
- `allow-set-microphone-device`
- `allow-start-session`
- `allow-stop-session`
- `allow-get-mic-muted`
- `allow-set-mic-muted`
- `allow-get-state`
